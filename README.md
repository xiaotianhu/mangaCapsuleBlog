## 🚀 关于漫画胶囊

**漫画胶囊** 是一款专为 iOS/iPadOS 打造的极简、现代且功能强大的本地漫画阅读器。它不仅是一个查看工具，更是漫画爱好者的“数字收藏柜”。

### ✨ 核心特性

- **🎨 AI 实时超分 (Real-CUGAN)**：内置业界领先的 AI 算法，能将低分辨率的旧漫画实时放大、降噪、锐化，在 Retina 屏幕上焕发新生。
- **📦 全格式兼容**：完美支持 ZIP, CBZ, RAR, CBR, EPUB, MOBI, AZW3, PDF 等主流漫画与电子书格式。
- **🔌 灵活的文件导入**：
  - 支持 **WebDAV** 远程挂载（NAS 玩家福音）。
  - 支持 **百度网盘**、**阿里云盘**、**Google Drive**、**OneDrive**、**Dropbox** 等主流云存储直接导入。
  - 支持 Wi-Fi 传输及系统文件分享。
- **📖 沉浸式阅读体验**：
  - 自由切换横屏/竖屏、单页/双页模式。
  - 智能切边、去除黑边，最大化利用屏幕空间。
  - 极速索引，即使是包含数千张图片的超大文件也能秒开。
- **🛡️ 隐私与安全**：无广告、无跟踪，所有处理均在本地完成，支持 FaceID/TouchID 应用锁。
