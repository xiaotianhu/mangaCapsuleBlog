WebDAVは現在のところ互換性が最も高く、性能が最もバランスが取れたファイル転送プロトコルの一つです。Manga Capsuleを組み合わせることで、公司でサボりながらも自宅NAS上の漫画を読むことができます（嘘）。

## 準備作業

まず、NASでWebDAVサービスが有効になっていることを確認し、ルーターでポート転送を設定してください（外部アクセスが必要な場合）。

```
// 典型的なWebDAV URL構造
http://[あなたのドメインまたはIP]:[ポート]/[共有フォルダパス]
```

Manga Capsuleの「接続」ページでWebDAVを選択し、上記の情報を入力します。接続に成功すると、ファイルリストがすぐに表示されます。
