# Manga Capsule - 究極のローカル漫画読書体験
[English](README.md) | [简体中文](README_zh.md)

![Manga Capsule Hero](./hero-3in1.png)

Manga Capsuleは、iOS/iPadOS向けに深くカスタマイズされたローカル漫画リーダーです。「大量のローカル漫画コレクションを優雅に管理し読書する方法」という課題に取り組んでいます。最先端のAI技術と極限的なインタラクティブデザインにより、漫画愛好家のための純粋で効率的なデジタルブックシェルフを実現します。

---

## 💎 Manga Capsuleを選ぶ理由

### 1. 🎨 エンドツーエンドAIアップスケーリング
多くの貴重な古い漫画は解像度が非常に低く、現代のハイビジョン画面ではぼやけてノイズが多く見えます。
- **リアルタイム処理**: 深く最適化されたAIアルゴリズムを内蔵。事前処理は不要で、ページをめくると同時に2x〜4xのアップスケーリングとノイズ除去が完了します。
- **視覚的な飛躍**: JPG圧縮アーティファクトを効果的に除去し、線をシャープにし、低解像度漫画に「リマスターされた」高画質视觉效果を与えます。
- **完全なローカル処理**: すべてのAI計算はデバイス上でローカル実行（Apple Neural Engineで加速）。プライバシー確保とデータ消費ゼロを実現します。

### 2. 🔌 マルチプロトコルリモートマウントと同期
モバイルデバイスのストレージ容量に制約されなくなります。
- **NASプレイヤーの夢**: **WebDAV**と**SMB**プロトコルを深くサポート。NASやパーソナルサーバーから直接大量のリソースを読み込め、ローカルファイルと同等のスムーズさを体験できます。
- **フルクラウドドライブサポート**: **Baidu Netdisk**、**Aliyun Drive**、**Google Drive**、**Dropbox**などをワンクリックでマウント。オンデマンドでクラウド漫画を読みられます。
- **极限同期**: Wi-Fi AirDrop、システムファイル共有、iCloud Driveインポートをサポート。
- **Komgaサポート**: OPDSベースのKomgaサーバーをサポートし、漫画管理がより簡単に。

### 3. 📦 豊富なフォーマットサポート
古いアーカイブでも最新の電子書籍に対応。
- **アーカイブ**: ZIP、CBZ、RAR、CBR。
- **電子書籍**: EPUB、MOBI、AZW3、PDF。
- **画像フォーマット**: JPG、PNG、WEBP、HEIC、GIFなど。

### 4. 📖 「没入感」のために生まれたインタラクティブデザイン
- **スマートクロップping**: スキャンされた漫画の余白を自動認識・クロップし、画面表示領域を最大化。
- **超高速レンダリング**: 自行開発のスライスレンダリングエンジンにより、数千枚の画像を含む超大ファイルでも瞬時のオープンとスムーズなスクロールを実現。
- **マルチモード**: ランドスケープ見開きページ（物理書籍体験の再現）、縦スクロール、日本式/西洋式の読み順切り替えをサポート。
- **精美なブックシェルフ**: カバーを自動抽出し、マルチレベルフォルダ管理をサポートし、独自の2次元ウォールを構築できます。
