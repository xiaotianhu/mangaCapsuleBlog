## How to Read Local Comics with Manga Capsule - Comic Import Tutorial

Manga Capsule supports free comic reading, allowing you to import local comics for reading. Currently supported import methods include:

- WiFi Transfer
- iCloud File Import
- Download from NAS
- USB Wired Import

Details are as follows:

#### WiFi Transfer
Tap the plus icon in the top right corner of the bookshelf, select WiFi Transfer, and grant local network permissions;
Requires iPhone/iPad and computer to be on the same WiFi network;
After opening, a URL will be displayed. Visit this URL on the computer you want to transfer books from, and follow the on-screen instructions to import.

#### iCloud File Import
Tap the plus icon in the top right corner of the bookshelf, select Import Files/Import Folder
In the pop-up window, find your iCloud directory or files, select them, and confirm to import

#### Download from NAS
After adding a NAS drive, you can long-press the book you want to download on the bookshelf (multi-select supported), and select Download from the pop-up menu.
After confirming, it will be added to the download queue and start downloading automatically.
You can view download queue tasks from Settings - Downloads.

#### USB Wired Import
Connect your iPhone/iPad to your Mac computer
In Finder, select the corresponding device
You can see the Manga Capsule app on the right side;
Simply drag the books you want to import directly onto the Manga Capsule app icon - no need to worry about specific folders, as they will be imported automatically after dragging.
