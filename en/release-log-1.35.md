## Manga Capsule 1.35 Released~
We've received a lot of feedback from our users, so here's an update addressing those issues — thank you all!  
- Vertical comics now support right-swipe to close gesture  
- File move operations now include a "New Folder" feature for better organization  
- Mobile Sidebar can now be summoned with a left-swipe gesture  
- Fixed the reading order for top-to-bottom comics, merging the next page correctly  
- Fixed the issue where the sidebar member card display toggle wouldn't save  
- Fixed the Baidu Cloud refresh issue — pull-to-refresh now updates the directory  
- Fixed the issue where special characters in Baidu Cloud folder names wouldn't display properly
