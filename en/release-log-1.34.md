## Manga Capsule 1.34 Released~

Major update for the free manga reader: OPDS protocol support now available
I've been traveling lately looking for inspiration, so updates have been a bit slow, but I've received all your feedback and continue to make improvements. There have been quite a few requests for OPDS protocol support. To be honest, I don't use it myself, and it took me several days just to research this thing.
	
I've gone back and forth on this feature before, giving up halfway each time—it's really not easy to implement. For someone like me who insists on hand-crafting apps the old-fashioned way, it was quite a dilemma. But after some thought, since we are an app that supports streaming reading, OPDS should be a pretty significant differentiator from our competitors. Let's do it!
	
So this major update finally brings OPDS support!
	
But... for now only Komga is supported. Other services still need adaptation. If you need support for other services, please contact us to provide feedback. Every OPDS server implementation is different and requires individual adaptation, so there's still some work to do, my apologies~
	
Some other changes:
- Volume button page turning is now disabled by default—can be enabled in 「General Settings」
- 「General Settings」 now includes an option to hide membership cards
- Fixed an issue where the merged next page size was incorrect
	
Feel free to try it out and share your feedback. If you run into any issues, don't hesitate to reach out.   Much gratitude～
