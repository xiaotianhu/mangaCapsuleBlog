    2025 has passed, and it was my first year of full-time development.

    Started large-scale testing in April, gathered a batch of beta testers who provided feedback on countless issues, most of which were resolved;

    Started promotion in September, got heavily criticized by many people;

    Continued fixing bugs and solving problems, reputation gradually improved, App Store rating reached 4.6, and gained dozens of positive reviews and many organic fans.

    Started earning some income, received first payment from Apple, though can't fully support myself yet, but it strengthened my determination to move forward.

    Looking at the git commit history, hoping to be more diligent in the new year, striving to bring more useful features and create more value~

    * 2025-01: 

     Completed development of VIP purchase flow, redemption code system, and multi-language switching, and integrated Mixpanel analytics.

   * 2025-02:

     Focused on iPhone version adaptation and Sidebar development, optimized bookshelf loading and "next book" auto-navigation logic.

   * 2025-03: Added Aliyun Drive support, optimized Mobi format parsing, and initially implemented simulated page turning effect and bookshelf UI refactoring.

   * 2025-04: Continued polishing simulated page turning experience and reader settings, supported WebDAV HTTP protocol, and optimized purchase flow and dark mode.

   * 2025-05: Deeply implemented custom PDF parsing engine (supporting multiple encodings and encryption), and added keyboard shortcut support.

   * 2025-06: Implemented AI super-resolution (Waifu2x) function and optimized memory usage, improved PDF rendering strategy, and added RAR archive support.

   * 2025-07: Launched App icon change function, improved PDF reader gestures (zoom, menu)

   * 2025-08: Fully implemented new user guide flow, added reader "merge next page" and zoom functions, and revamped VIP interface.

   * 2025-09: Improved new user guide and coupon/redemption code system, implemented system language following

   * 2025-10: Added Dropbox and Google Drive cloud storage support, optimized pull-to-close gesture

   * 2025-11: Focused on implementing smart page splitting function (automatic detection and splitting of double-page images), and started development of OPDS protocol support.

   * 2025-12: Improved OPDS bookshelf and reader support, launched new official website and blog, and performed a series of year-end stability fixes.
