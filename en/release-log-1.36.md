## Manga Capsule Version 1.36 Released~

The Lunar New Year has ended, but we weren't idle during the holidays either.

It's been quite a while since the last version, because we've made many fundamental changes—after all, a solid foundation is essential for future development. So this release took longer than usual, but it brings a wave of optimizations. Let's get better together in the new year~

- Disks now support renaming and drag-to-sort
- Disks can now reselect their root directory path
- Added "text-free mode" for the bookshelf
- AZW3 format is now supported
- Reader now supports swipe-left to go back
- Fixed sorting and missing page issues with Mobi format
- Added individual book cache clearing (in the long-press menu on bookshelf)
- Bookshelf now supports filtering to show read/unread books

Feel free to try it out and share your feedback. I'm always here if you have any questions. Thanks for your support～
