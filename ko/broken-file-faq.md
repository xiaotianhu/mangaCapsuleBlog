## 파일 손상 일반적인 원인

####百度HD 가져오기
百度HD에서 파일을 가져오려면百度HD에서 파일을 다운로드하고 «다른 앱으로 열기»를 선택해야 합니다.
«파일»에서 파일을 선택하여 Manga Capsule으로 직접 가져오면 파일을 열 수 없습니다. 이는 전체 파일이 아닌 바로 가기 링크이기 때문입니다.百度HD에서 가져와야 합니다.

#### 클라우드 스토리지 공유
百度网盘,阿里云盘 등 클라우드 스토리지에서 Manga Capsule으로 파일을 공유할 때 «다른 앱으로 열기»를 통해 공유해야 합니다. 파일 다운로드 과정이 표시되면 공유된 파일이 완전한 올바른 파일입니다.
百度HD의 공유 기능은 바로 가기 링크를 공유하므로 전체 파일이 아니어 열 수 없습니다.

#### 다운로드 실패
일부 파일은 다운로드 과정에서 네트워크 문제로 완전히 다운로드되지 않고 종료될 수 있습니다. 이러한 파일은 이미 손상되어 열 수 없습니다. 다시 다운로드해 보세요.

#### 암호화된 파일
Manga Capsule은 현재 암호화된 압축 파일 읽기를 지원하지 않습니다. zip/rar 등의 파일이 암호화되어 있으면 먼저 해독한 후 찾아볼 수 있습니다.

#### 중첩 압축 파일
Manga Capsule은 현재 중첩된 압축 파일을 지원하지 않습니다. 즉, 압축 파일 안에 다른 압축 파일이 있거나 압축 파일 안에 epub/mobi/pdf 등의漫画 파일이 있는 경우는 지원되지 않습니다. 먼저 압축을 푼 후 읽어야 합니다.
