    2025년이 지나갔고, 저는 본격적으로 개발에 뛰어든 첫 해였습니다.

    4월부터 대규모 테스트를 시작하여 많은 베타 팬들을 확보했고, 무수한 문제들을 피드백받았으며, 대부분 해결되었습니다;

    9월부터 정식 출시 및 홍보를 시작했는데, 많이 비판받았습니다;

    계속해서 버그를 수정하고 문제를 해결하면서 평판이 점차 개선되기 시작했습니다. Appstore 평점은 4.6이며, 여러 좋은 리뷰와 자연스러운 팬들을 확보했습니다.

    점차 수입이 생기기 시작했습니다. 처음으로 애플에서 급료를 받았는데, 아직 자립할 수는 없지만 더 나아갈决心이 더 강해졌습니다.

    git 커밋 기록을 살펴보니, 새 해에는 더 열심히 달려서 여러분에게 더 많은 유용한 기능을 제공하고 가치를 실현하고 싶습니다~

    * 2025-01: 

     VIP 구매 프로세스, 교환코드 시스템 및 다국어 전환 기능을 완성하고, Mixpanel 데이터 분석을 연동했습니다.

   * 2025-02:

     iPhone 버전 호환 및 사이드바(Sidebar) 개발에 중점을 두고, 책장 로딩과 "다음 책" 자동 이동 로직을 최적화했습니다.

   * 2025-03:阿里云盘(Aliyun Drive) 지원을新增하고, Mobi 형식 파싱을 최적화하며, 시뮬레이션 페이지 넘김 효과와 책장 UI 재구성을初步实现했습니다.

   * 2025-04: 시뮬레이션 페이지 넘김 경험과 리더기 설정을 지속적으로 다듬고, WebDAV HTTP 프로토콜을 지원하며, 구매 프로세스와 다크 모드를 최적화했습니다.

   * 2025-05: 다양한 인코딩과 암호화를 지원하는 사용자 정의 PDF 파싱 엔진을底层에서 구현하고, 키보드 단축키 지원을 추가했습니다.

   * 2025-06: AI 초해상도(Waifu2x) 기능을实现하고 메모리占用을 최적화하며, PDF 렌더링 전략을 개선하고 RAR 압축 파일 지원을新增했습니다.

   * 2025-07: 앱 아이콘 변경 기능을 출시하고, PDF 리더기 제스처(확대, 메뉴)를 완성했습니다

   * 2025-08: 초보자 가이드 프로세스를 전면 구현하고, 리더기 "다음 페이지 합치기"와 확대 기능을新增하며, VIP 인터페이스를 개편했습니다.

   * 2025-09: 초보자 가이드와 쿠폰/교환코드 체계를 완성하고, 다국어가 시스템을 따르도록 구현했습니다

   * 2025-10: Dropbox와 Google Drive 클라우드 저장소 지원을新增하고, 아래로 닫기 제스처를 최적화했습니다

   * 2025-11: 스마트 페이지 분할 기능(자동 인식 및 이중 페이지 이미지 분할)을重点实现하고, OPDS 프로토콜 지원 개발을 시작했습니다.

   * 2025-12: OPDS 기능의 책장과 리더기 지원을 완성하고,全新的 공식 사이트와 블로그를 출시하며, 연말 안정성 수정一系列을 진행했습니다.
